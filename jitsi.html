An easy to use video-conference application on your browser

Today I used meet.jitsi.si, a simple to use interface on my browser. I experienced that while I was sharing my desktop, my screen was producing fractal images. I am unsure if that happens on other applications like google's hangouts. Running on Arch Linux I found online an AUR package, but I realized that this is messenger application.

I downloaded this package because I was not able to make a valid conference room in the first place, though a friend next to me found out that the problem was an underscore that I had typed on the URL (the room's name). That was redirecting my browser to an error page saying something about Ubuntu, and I thought in the first place that jitsi was unable to identify my OS. By removing the underscore the problem was solved and the room was working.
